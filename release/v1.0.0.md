# dottracker v1.0.0

dottracker is a tracking program for UNIX dotfiles and configurations.

## Synopsis

`dottracker [-v | --version] [-h | --help] <command> [<args>]`

## Enviroment

The following enviroment variables must be exported for the program to run
correctly:

- `DOTTRACKER_ARCHIVE`: Required, represents the location of the archived
  files.
- `DOTTRACKER_CONFIG`: Defaults to '$HOME/.config/dottracker/files.conf',
  represents the location of the configuration file. See 'files.conf' for more
  information about configuration options.

## Commands

Name        | Description
:---------- | :-----------
`diff`      | Print a diff of the tracked files
`untrucked` | Print a list of the archived and not tracked files
`update`    | Update files in target
